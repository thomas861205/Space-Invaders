# Space-Invaders 🚀

The classic arcade game Space Invaders implemented on a FPGA written in Verilog.

Featuring:
- Monitor display (VGA port) 📺
- Keyboard control ⌨️
- 2 playable stages 👾
- HP display with LED 🩸
- Score display with 7 segment display 🏆
- Cheat code (type "Invincible") 😎
